# S-008: Team Icebreaker Settings and Custom Question Management

## User Story

As a **team admin**, I want to **configure whether icebreakers are enabled, set a default category, manage custom questions, and optionally set an auto-timer** so that **I can customize the warmup experience to match my team's culture and preferences**.

## Context

Not every team wants icebreakers. Some teams are already tight-knit and want to jump straight to writing. Some teams want icebreakers every time but only from the "Quick" category. Some teams have inside jokes that make great custom questions. This story gives team admins the controls to make the icebreaker theirs.

## Acceptance Criteria

### Settings

- [ ] Team settings include an "Icebreaker" section (accessible from team settings page)
- [ ] **Enable/disable toggle**: when disabled, new boards skip icebreaker phase and start in `write` directly
- [ ] **Default category**: optional â€” if set, new icebreakers auto-filter to this category (All, Fun, Team-Building, Reflective, Creative, Quick)
- [ ] **Auto-timer**: optional â€” if set (e.g., 3 minutes), a countdown timer appears during icebreaker phase. When it reaches 0, a gentle "Time's up!" notification appears (does NOT auto-transition â€” facilitator still controls the transition)
- [ ] Settings are stored in a `team_settings` JSONB column or dedicated table
- [ ] Settings changes are team-admin-only

### Custom Questions

- [ ] Team admins can view all custom questions created for their team
- [ ] Team admins can add new custom questions (question text + category)
- [ ] Team admins can delete custom questions they created
- [ ] Custom questions appear alongside system questions when generating icebreakers (already implemented in service)
- [ ] Custom question form is in the team settings page (not on the icebreaker screen â€” keep the warmup screen clean)
- [ ] Custom questions show who created them and when

## Implementation Notes

### Database

New migration:

```sql
-- Add icebreaker settings to teams table
ALTER TABLE teams ADD COLUMN icebreaker_enabled BOOLEAN NOT NULL DEFAULT true;
ALTER TABLE teams ADD COLUMN icebreaker_default_category VARCHAR(20);
ALTER TABLE teams ADD COLUMN icebreaker_timer_seconds INTEGER;

-- Constraints
ALTER TABLE teams ADD CONSTRAINT chk_icebreaker_category
  CHECK (icebreaker_default_category IS NULL OR icebreaker_default_category IN ('fun', 'team-building', 'reflective', 'creative', 'quick'));
ALTER TABLE teams ADD CONSTRAINT chk_icebreaker_timer
  CHECK (icebreaker_timer_seconds IS NULL OR (icebreaker_timer_seconds >= 30 AND icebreaker_timer_seconds <= 600));
```
