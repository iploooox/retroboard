# S-025: Advanced Templates (4Ls, Mad/Sad/Glad, Sailboat, Starfish)

## User Story

As a **facilitator**, I want to **choose from a variety of advanced retro templates** so that **I can use different formats to keep retrospectives engaging and suitable for different team situations**.

## Acceptance Criteria

- [ ] Four additional templates are available: 4Ls, Mad/Sad/Glad, Sailboat, Starfish
- [ ] 4Ls template has columns: Liked, Learned, Lacked, Longed For
- [ ] Mad/Sad/Glad template has columns: Mad (red), Sad (blue), Glad (green)
- [ ] Sailboat template has columns: Wind (propelling), Anchor (holding back), Rocks (risks), Island (goals)
- [ ] Starfish template has columns: Keep Doing, More Of, Less Of, Stop Doing, Start Doing
- [ ] Each template includes a description and recommended use case
- [ ] Templates appear in the template selection grid alongside basic templates
- [ ] Each template has a distinct visual preview showing its column layout and colors
- [ ] Templates are seeded via database migration (system templates)
- [ ] Template metadata includes suggested phase timer durations

## Implementation Notes

- Same database structure as S-012 (templates + template_columns tables)
- Add new seed records for each advanced template
- Sailboat template is metaphor-based; column descriptions should explain the metaphor
- Starfish has 5 columns -- UI may need to handle wider layouts
- Consider adding template category field for grouping (basic, advanced, metaphor-based)
