# S-004: Invite Members via Shareable Link

## User Story

As a **team admin or facilitator**, I want to **generate a shareable invite link for my team** so that **new members can join easily without manual account setup**.

## Acceptance Criteria

- [ ] Team admin or facilitator can generate an invite link for their team
- [ ] Invite links contain a unique, unguessable token (UUID v4 or similar)
- [ ] Invite links have a configurable expiration (default: 7 days)
- [ ] Invite links can optionally set the role for joiners (default: member)
- [ ] Clicking an invite link while logged in adds the user to the team
- [ ] Clicking an invite link while logged out redirects to register/login, then auto-joins
- [ ] Team admin can view active invite links and revoke them
- [ ] Expired or revoked invite links return a clear error message
- [ ] Maximum of 5 active invite links per team at any time
- [ ] Invite link usage count is tracked

## Implementation Notes

- Store invites in `team_invites` table: id, team_id, token, role, created_by, expires_at, revoked_at, use_count, max_uses
- Invite link format: `{APP_URL}/invite/{token}`
- When a user follows an invite link, the frontend calls `POST /api/v1/teams/join` with the token
- Consider adding max_uses field for limited-use links
