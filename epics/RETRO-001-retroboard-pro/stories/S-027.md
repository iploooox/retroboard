# S-027: Board Color Themes per Team

## User Story

As a **team admin**, I want to **customize the board color theme for my team** so that **our retrospective boards feel personalized and visually distinct**.

## Acceptance Criteria

- [ ] Team admin can select a color theme from a set of pre-defined themes
- [ ] Available themes include: Default, Ocean, Sunset, Forest, Midnight, Lavender, Coral, Monochrome
- [ ] Theme affects board background, column headers, card backgrounds, and button colors
- [ ] Theme is applied team-wide (all boards for the team use the same theme)
- [ ] Theme changes take effect immediately for all connected participants
- [ ] Theme selection is accessible from team settings
- [ ] Each theme has a preview swatch/thumbnail in the selection UI
- [ ] Theme preference is persisted and loaded on board access
- [ ] Themes maintain WCAG AA contrast ratios for text readability
- [ ] Individual users can override the team theme with a personal preference (optional)

## Implementation Notes

- Add `theme` column to `teams` table (string, default: 'default')
- Define theme definitions as CSS custom properties (variables)
- Each theme: { name, primary, secondary, background, surface, text, accent, columnColors[] }
- Apply theme via CSS custom properties on the board container
- Broadcast theme change via WebSocket: `team:theme_changed`
