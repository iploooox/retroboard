# S-028: Icebreaker Generator

## User Story

As a **facilitator**, I want to **generate a random icebreaker question to start the retro** so that **the team warms up, feels comfortable, and the retrospective begins on a positive note**.

## Acceptance Criteria

- [ ] Facilitator can generate a random icebreaker question from a curated library
- [ ] Icebreaker library contains at least 50 questions across categories
- [ ] Categories include: Fun, Team-Building, Reflective, Creative, Quick
- [ ] Facilitator can filter by category
- [ ] Facilitator can skip/re-roll to get a different question
- [ ] Selected icebreaker is displayed to all board participants in real-time
- [ ] Facilitator can dismiss the icebreaker display when done
- [ ] Icebreaker history is tracked per team (avoid repeating recent ones)
- [ ] Custom icebreaker questions can be added by team admin
- [ ] Icebreaker feature is optional and can be skipped

## Implementation Notes

- `icebreakers` table: id, question, category, is_system, created_by, created_at
- `team_icebreaker_history` table: id, team_id, icebreaker_id, used_at
- Seed 50+ system icebreaker questions across categories
- Random selection: exclude recently used (last 10) for the team
- Display via WebSocket broadcast: `icebreaker:shown`, `icebreaker:dismissed`
